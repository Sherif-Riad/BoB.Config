Per Dino Settings Top ↑
Creature Group Soft-Limit
Population (%) Sets a server population cap for this specific creature relative to the total server population.
For example, setting Pteranodon to 50% means that no more than 50% of the server population can consist of Pteranodons.
Exceeding this limit will result in a significantly reduced growth rate for those creatures.
Growth Limit Sets the maximum growth a creature can achieve.
This value is hard-capped at 50.
Soft-Limit Min Defines the growth level at which new group members begin triggering the overpacked debuff.
Soft-Limit Max Indicates the growth level at which new group members trigger the maximum strength of the overpacked debuff.
This page requires JavaScript to function properly. Please enable JavaScript in your browser settings and reload the page.

Admin Rank Settings Top ↑
Rank Name This is the server-side name used for setting permissions, not the tag that will appear next to the admin's name in chat.
Only A-Z and a-z characters are allowed.
Each rank name must be unique.
Rank Level The numeric level of the rank, ordered in ascending sequence.
Each rank must have a unique level number.
Ranks with higher level numbers inherit the permissions of those with lower levels.

Steam ID This is the player's SteamID64.

Staff Rank
Chat Tag This tag appears before the name in chat. Use it to identify Staff, VIPs, or special individuals.

Name/Tag Color Colors the user's tag and name, without affecting chat color.
Click the color to modify it.
Accepts RGB, HSV, or Hex values.
Leave as White to not set a color.
The value will be adjusted to the closest accepted by the game.
Color Hex This is the Hexcode for the Name/Tag Color.
